1.主要解决移动端适配 2.引入的是淘宝自适应方案(网址：  https://github.com/amfe/lib-flexible ; )
定义函数：
var getDpr = function getDpr(){
    var dpr = $('html').attr('data-dpr');
    if (dpr == 1) {
      return 12;
    }else if (dpr == 2) {
      return  24;
    }else {
      return 36;
    }
};
调用方法：fontSize:getDpr()



以下是实例：
label: {
    normal: {
        formatter: 'y = 0.5 * x + 3',
        textStyle: {
            align: 'right',
            color:"green",
            fontSize:getDpr()
        }
    }
},
