vue测试页面
按照官网配置好appid,AppSecret,IP白名单,调试时发现跨域问题
1.在config下的index.js中的dev里配置：
// 跨域
proxyTable: {
    '/api': {
        target: 'https://api.weixin.qq.com/cgi-bin/token',
        changeOrigin: true,
        pathRewrite: {
            '^/api': ''
        }
    }
}
2.main.js：
// 跨域
Vue.prototype.WX = '/api'
3.ajax调用：
let url = this.WX
AjaxPlugin.$http.get(url,{
    params:{
        grant_type:'client_credential',
        appid:AppID,
        secret:AppSecret
    }
})
.then((response)=>{
    console.log('access-token',response.data)
})
结束。
