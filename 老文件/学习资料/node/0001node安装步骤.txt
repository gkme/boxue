1. https://nodejs.org/en/ 下载node 并安装到D盘nodejs文件下(建议不要用c盘)
2.摁住shift并点鼠标右键 点击 在此处打开命令行窗口(w) 
    输入 node -v 回车查看是否安装node成功
    输入 npm -v 回车查看是否安装npm成功
3.在nodejs文件夹下创建  node_cache文件夹 node_global文件夹
4.命令行窗口 npm config set prefix "D:\nodejs\node_global"
    npm config set cache "D:\nodejs\node_cache"
5.“我的电脑”右键“属性”-“高级系统设置”-“高级”-“环境变量”
在用户变量下 PATH 的变量值修改为：C:\Program Files\nodejs\node_global
在系统变量下 新建 变量名: NODE_PATH  变量值: C:\Program Files\nodejs\node_global\node_modules
  
6.npm install -g cnpm --registry=https://registry.npm.taobao.org 安装淘宝镜像
7.cnpm install express-generator -g 全局安装应用生成器  注：非必须

测试环节：
在新建文件夹下 cnpm init
创建 server.js 并输入一下内容
var http = require("http");
http.createServer(function(req,res){
    res.writeHead(200,{"Content-Type":"text/plain"});
    res.end("hello");
}).listen(8888);
npm start 命令行启动服务
在浏览器：localhost:8888  即显示：hello
恭贺你：安装成功
