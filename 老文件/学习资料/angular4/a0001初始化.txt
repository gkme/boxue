npm install -g @angular/cli@1.4.10  用脚手架下载angular4(此处加了版本限制)
ng new myapp  创建一个myapp的文件夹
cd myapp 进入myapp文件夹
npm i  下载依赖

三、第三方库的安装
    引入jq和bootstrap：https://blog.csdn.net/LanTingShuXu/article/details/78886635 参考
    npm i jquery@1.11.3 --save
    npm i bootstrap@3.3.7 --save
    .angular-cli.json文件：
    "styles":中添加
    "../node_modules/bootstrap/dist/css/bootstrap.min.css"
    "scripts":中添加
    "../node_modules/jquery/dist/jquery.min.js",
    "../node_modules/bootstrap/dist/js/bootstrap.min.js"
    typeScript添加上类型声明：
    // 添加这个是为了让typeScript认识$符号
    npm install @types/jquery --save-dev
    // 添加上bootstrap的声明
    npm install @types/bootstrap --save-dev
四、创建组件
    ng g component 组件名


<div class="row" style="margin:0;">
  <div class="col-md-3">左边</div>
  <div class="col-md-3">中间</div>
</div>

ng server -o  开启服务
